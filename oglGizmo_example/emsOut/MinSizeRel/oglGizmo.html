Actual virtualGizmo usage (but freely configurable) :

Left mouse button -> free rotation
Shft + Left mouse button -> rotation ONLY around X
Ctrl + Left mouse button -> rotation ONLY around Y
Alt oR Super + Left mouse button -> rotation ONLY around Z
Right mouse button -> Pan
Shift or Ctrl + Right mouse button -> Dolly/Zoom
Left click -> stop idle rotation

(some browser have a trouble with right button, for internal use)
